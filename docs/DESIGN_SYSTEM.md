# Design System

## Paleta
- Primary: definido por tenant (`tenants.json`).
- Neutros: Tailwind `slate`.

## Tipografía
- Inter (fallback sans-serif).

## Componentes reutilizables
- AppShell (sidebar/topbar dinámico)
- StatCard
- RiskBadge
- MasteryBar
- ActivityCard
- NotificationBell
- Heatmap simple
- DataTable
- EmptyState
- LoadingSkeleton
- WizardStep

## Layout
- Grid responsivo con Tailwind.
- Angular Material para navegación y tablas.
